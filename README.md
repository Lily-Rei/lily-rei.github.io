# lily-rei.github.io

Personal site and technical archive for Lily Rei.

This repository powers my GitHub Pages site, where I publish cybersecurity content focused on hands-on learning, Capture the Flag development, offensive security workflows, lab environments, and student-focused technical training.

## What This Site Is

This is not just a landing page.

It is a living archive for:
- CTF write-ups
- custom challenges
- lab notes
- technical walkthroughs
- training material
- resources for students and practitioners

The goal is to build content that helps people move beyond memorization and into real understanding.

## Focus Areas

- Offensive security
- Capture the Flag challenge design
- Enumeration and reconnaissance
- Privilege escalation
- Pivoting and internal movement
- Web exploitation
- Active Directory concepts
- Cyber range and lab development
- Practical cybersecurity education

## Site Structure

- `index.md` — homepage
- `writeups.md` — CTF write-ups and walkthroughs
- `challenges.md` — original challenge concepts and exercises
- `labs.md` — lab notes, builds, and attack paths
- `resources.md` — guidance, references, and student material
- `_layouts/` — custom layout overrides
- `_data/navigation.yml` — sidebar navigation
- `assets/css/style.scss` — theme overrides and custom styling

## Theme

This site uses the GitHub Pages remote Hacker theme:

```yml
remote_theme: pages-themes/hacker@v0.2.0
plugins:
  - jekyll-remote-theme
```

## Philosophy

Cybersecurity is learned by doing.

This project exists to document process, preserve lessons learned, and create practical material that helps others think more critically, work more methodically, and understand not just what works, but why it works.
